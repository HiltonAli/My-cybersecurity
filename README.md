# My-cybersecurity basics notes
 Introduction to Cybersecurity

Cybersecurity is the practice of protecting systems, networks, and data from attacks.

CIA Triad

The CIA Triad is the foundation of cybersecurity:

Confidentiality = Only authorized users can access data.
Integrity = Data cannot be altered without permission.
Availability = Systems and data are accessible when needed.


MFA (Multi-Factor Authentication)
Extra security for your accounts.
Example: Password - Phone code

Networking Basics
IP Address - A device's address on a network.
Router - Connects devices to the internet.
Firewall - Blocks unwanted traffic.
